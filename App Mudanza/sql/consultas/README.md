# Consultas SQL – Reportes y Búsquedas

En esta carpeta se encuentra un conjunto de consultas diseñadas para obtener información útil desde la base de datos del sistema de mudanzas.

## Contenido

### 03_consultas_reportes.sql
Incluye consultas para:

- Mostrar todas las reservas de un cliente
- Buscar reservas por fecha
- Contar reservas por cliente
- Contar reservas por tipo de servicio
- Listar las próximas mudanzas ordenadas por fecha y hora

## Cómo ejecutarlo

1. Abrir el archivo en MySQL Workbench.
2. Seleccionar una consulta específica o ejecutar todo el archivo.
3. Revisar los resultados en la sección inferior del editor.

Estas consultas permiten analizar los datos y verificar que la estructura de la base de datos funciona correctamente.
