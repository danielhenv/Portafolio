# DDL – Definición de la Base de Datos

En este directorio se encuentra el archivo con las instrucciones necesarias para crear la base de datos del proyecto App Mudanza.

## Contenido

### 01_creacion_tablas.sql
Incluye:

- Creación de la base de datos `mudanza_db`
- Tabla `Cliente`
- Tabla `Reserva`
- Definición de claves primarias
- Relación mediante clave foránea entre ambas tablas

## Cómo ejecutarlo en MySQL Workbench

1. Abrir MySQL Workbench.
2. Crear una nueva pestaña de consulta.
3. Cargar el archivo o copiar su contenido.
4. Ejecutar el script completo con el botón del rayo.

Después de ejecutarlo, las tablas deberían aparecer dentro del esquema `mudanza_db`.
