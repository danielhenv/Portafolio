# DML – Manipulación de Datos

Este directorio contiene el archivo con ejemplos de inserción, actualización y eliminación de datos para probar la base de datos creada para App Mudanza.

## Contenido

### 02_datos_ejemplo.sql
Incluye:

- Inserción de clientes
- Inserción de reservas
- Actualización de campos específicos
- Eliminación de una reserva de ejemplo

## Cómo usarlo

1. Asegurarse de haber ejecutado primero el archivo DDL.
2. Abrir este archivo en MySQL Workbench.
3. Ejecutar todo el script o únicamente las líneas necesarias.
4. Consultar las tablas para verificar los cambios.

Este archivo sirve como base para llenar la base de datos y validar su funcionamiento.
